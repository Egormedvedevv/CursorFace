# CursorFace

Интерактивный веб-сайт, где персонажи следят за курсором или касанием на экране.

## Технологии

- React 18
- Vite
- React Router

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Функции

- Интерактивные лица, которые следуют за курсором/касанием
- Анимация глаз, следящих за курсором
- Переключение между 6 персонажами
- Адаптивный дизайн для мобильных устройств
- Поддержка touch-событий для мобильных устройств
