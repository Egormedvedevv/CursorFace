CursorFace
CursorFace is a playful and slightly eerie web experiment where characters follow your every move. Built with HTML, CSS, and JavaScript, the site offers an animated, interactive experience — perfect for when you're bored and want a fun distraction.
Features include cursor-tracking characters, smooth transitions between them, a minimal retro-styled interface, and an info window with a humorous description.
Technologies used: HTML, CSS, JavaScript (vanilla).
This project exists just for fun — sometimes it's nice to be stared at by a weird little face.

CursorFace
CursorFace — это забавный и немного пугающий веб-эксперимент, где персонажи следят за каждым движением курсора. Сайт создан с использованием HTML, CSS и JavaScript и представляет собой анимированное интерактивное развлечение — особенно, когда скучно и хочется отвлечься.
Среди особенностей: персонажи, реагирующие на курсор, плавная смена образов, минималистичный интерфейс в ретро-стиле и информационное окно с шуточным описанием. Сайт неадаптивен.
Используемые технологии: HTML, CSS, JavaScript (без библиотек).
Этот проект создан просто для развлечения — иногда приятно, когда на тебя смотрит странное лицо.
